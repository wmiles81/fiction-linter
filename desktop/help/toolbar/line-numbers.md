## Line Numbers

Click **Line #s** in the toolbar (top center, in the view-toggle group) to show a numbered gutter along the left edge of the editor. Click **No #s** to hide it again.

### What the numbers count

Fiction Linter numbers *paragraphs*, not individual text lines. Each block of prose — whether it wraps across one screen line or ten — gets a single number. This maps to how writers think about their manuscripts and matches the word-level addressing used in the findings JSON sidecar.

### Scroll synchronization

The gutter scrolls in sync with the editor. The current paragraph is not specially highlighted, but the numbers provide a consistent visual reference as you move through a long chapter.

### Persistence

Your line-number preference is saved automatically in the app's local storage. The gutter state (on or off) is restored the next time you open Fiction Linter, even after a restart.

### Font size

The gutter numbers scale with the editor font size. If you increase or decrease the text size using the **A−** / **A+** buttons in the format bar, the gutter adjusts automatically.
