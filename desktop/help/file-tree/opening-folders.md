## Opening Folders

The **Files** panel on the left side of the window shows your manuscript's folder structure. To load a folder, click the **Open** button in the top-right corner of the Files panel header.

### Choosing a folder

Clicking **Open** shows the system folder picker. Select any folder on your computer — for example, the root folder of your manuscript project. Fiction Linter lists all files and subfolders within it.

You can also open a folder from the **File** menu: **File → Open Folder**.

### Persistence across restarts

The last folder you opened is remembered automatically. The next time you launch Fiction Linter, the same folder loads without you having to pick it again. If the folder has been moved or deleted since your last session, the file tree shows an empty state — click **Open** to choose a new location.

### Expanding and collapsing subfolders

Click any folder name in the tree to expand or collapse it. Expanded folders show a **▾** icon; collapsed folders show **▸**.

### Ancestor highlighting

When a file is open in the editor, all of its parent folders in the tree are highlighted with a subtle background tint. This gives you a visual breadcrumb trail so you can always see where the active file lives in the folder hierarchy, even when subfolders are deeply nested.
