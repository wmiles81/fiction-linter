## Bold, Italic, Underline

The **B**, *I*, and <u>U</u> buttons in the format bar (directly above the editor) apply inline formatting to selected text. You can also use keyboard shortcuts without reaching for the mouse.

### Keyboard shortcuts

| Format | Mac | Windows / Linux |
|--------|-----|----------------|
| Bold | **Cmd+B** | **Ctrl+B** |
| Italic | **Cmd+I** | **Ctrl+I** |
| Underline | **Cmd+U** | **Ctrl+U** |

### How to apply formatting

1. Select the word or phrase you want to format.
2. Click the button in the format bar, or press the keyboard shortcut.

Applying the same format a second time while the selection is already formatted removes it (toggles off).

### In the saved file

Bold and italic round-trip to standard markdown: `**bold**` and `*italic*`. Underline is preserved as an HTML `<u>` tag in the markdown, since plain markdown has no underline syntax. When you reopen the file, all three formats are restored correctly.

### Tips for fiction writers

In prose, bold and underline are rarely used in body text. Italic is the standard way to mark emphasis, titles of works, foreign words, and internal thoughts. Use formatting sparingly so it stays meaningful.
