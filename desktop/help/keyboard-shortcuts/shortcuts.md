## Keyboard Shortcuts

All shortcuts use **Cmd** on macOS and **Ctrl** on Windows/Linux.

### Global

| Shortcut | Action |
|----------|--------|
| F1 | Open Help |
| Cmd+, | Open Settings |
| Cmd+Shift+O | Open Folder |
| Cmd+N | New Tab |
| Cmd+W | Close Tab |
| Cmd+L | Toggle Lint |
| Cmd+Shift+L | Toggle Findings Display |

### Editor

| Shortcut | Action |
|----------|--------|
| Cmd+S | Save |
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
| Cmd+B | Bold |
| Cmd+I | Italic |
| Cmd+U | Underline |
| Cmd+A | Select All |

### Dialogs

| Shortcut | Action |
|----------|--------|
| Escape | Close dialog / dismiss popup |
| Enter | Activate (in License dialog) |
