## Decisions made

- **Decision:** LemonSqueezy for licensing, not self-hosted or Mac App Store
  - **Why:** Free plan ($0/mo), 5%+$0.50/transaction, no server to maintain. Multi-product support.
  - **Alternatives considered:** Self-hosted validation server (ops overhead), Mac App Store (30% cut, sandbox constraints), offline-only serial key (no revocation)

- **Decision:** Separate .findings.json and .annotation.md files (not merged)
  - **Why:** Different lifecycles — findings are regenerated on every save (machine snapshot), annotations are curated by the user (human notes). Merging would require parsing markdown back to distinguish auto vs manual entries.

- **Decision:** Help content as markdown with YAML frontmatter, not a database or JSON
  - **Why:** Same source serves both in-app help tab and external HTML manual. Markdown is writable by humans, parseable by machines, diffable in git.

- **Decision:** Context-sensitive help via right-click popup + "More..." link, not hover or help mode
  - **Why:** Hover is already used for lint tooltips. Help mode adds a modal state. Right-click is unambiguous and preserves the user's flow.

- **Decision:** Mac build as .zip not .dmg for v1.0
  - **Why:** dmg-builder has a Python compatibility issue on macOS 26. App works identically from a zip.

## Gotchas learned

- **electron must be in devDependencies** — electron-builder refuses to package if it's in dependencies (the binary is handled separately by the builder)
- **dmg-builder breaks on macOS 26** — Python 3.14 compat issue in dmgbuild. Use .zip target as workaround.
- **YAML frontmatter with colons** — a summary field containing "Cmd+B:" breaks js-yaml parsing. Quote the value.
- **CSS counters inside contenteditable** — Chromium suppresses ::before pseudo-elements on editable children in some cases. Use real DOM elements for line numbers.
- **React components don't forward data-* props** — `<ThemePicker data-help-id="x">` silently drops the attribute. Wrap in a `<div>` instead.
- **png-to-ico needs a 256px source** — the 2048px PNG generated an ICO that electron-builder rejected as "unknown format". Resize to 256px first.
